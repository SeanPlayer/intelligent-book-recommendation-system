# 基于Flask的智能图书推荐与管理系统

## 项目简介
本项目为毕业设计，旨在构建一个集成协同过滤、内容推荐和混合推荐策略的图书馆管理原型系统。系统基于Flask+Bootstrap+MySQL开发，实现了从用户管理、借阅流程到个性化推荐的完整闭环。

## 技术栈
- 后端：Python 3, Flask
- 前端：Bootstrap, HTML/CSS
- 数据库：MySQL（不包含在此仓库中）
- 推荐算法：协同过滤（UserCF/ItemCF）、内容推荐、加权混合

## 主要功能模块
- 用户管理：注册登录、权限控制、信誉分系统
- 图书管理：图书信息管理、库存状态
- 借阅管理：借阅归还流程、逾期处理
- 智能推荐：首页热门推荐、个性化推荐页、冷启动处理
